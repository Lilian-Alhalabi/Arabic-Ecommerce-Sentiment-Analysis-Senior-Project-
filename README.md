# Arabic E-Commerce Reviews Sentiment Analysis 🛍️

## Overview
An AI-powered system that classifies Arabic 
e-commerce reviews into Positive, Negative, 
and Neutral sentiments using MARBERT transformer model.

## Features
- Arabic text preprocessing and normalization
- MARBERT-based sentiment classification
- Interactive Streamlit dashboard
- Supports both Arabic and English interface
- Data balancing using RandomOverSampler

## Tech Stack
- Python
- HuggingFace Transformers (MARBERT)
- PyTorch
- Streamlit
- Scikit-learn
- Pandas

## Model Performance
| Metric | Score |
|--------|-------|
| Accuracy | 94.78% |
| Precision | 94.84% |
| Recall | 94.78% |
| F1-Score | 94.74% |

## Installation
pip install -r requirements.txt

## How to Run
streamlit run app/streamlit_app.py

## Dataset
Arabic Company Reviews Dataset from Kaggle
